# Voice Switcher

Voice Switcher is an NVDA add-on that helps you save and reload speech voice presets from a simple dialog. It is designed to be easy to use with a screen reader and to make it quicker to return to a preferred voice setup.

## What It Does

- Saves your current voice settings as a preset.
- Lets you load a saved preset whenever you need it.
- Helps you keep several voice setups available for different tasks.
- Stores your presets in your NVDA user configuration, so they are available again after restart.

## How To Install

1. Download the packaged `.nvda-addon` file.
2. In Windows Explorer, locate the add-on file and press `Enter`, or open NVDA and choose the option to install an add-on from a file.
3. Confirm the installation when NVDA asks.
4. Restart NVDA if prompted.

After installation, the add-on will be available the next time NVDA starts.

## How To Use

1. Press `NVDA+W` to open the Voice Switcher dialog.
2. Select **Save current setting** to store the voice you are currently using.
3. Type a name for the preset and confirm.
4. To use a preset later, open the dialog again, select the preset you want, and press `Enter` or choose **Load**.

You can also:

- Press `F2` to rename the selected preset.
- Press `Delete` to remove the selected preset.

## Tips

- If you regularly switch between voices, give each preset a clear name that will be easy to recognize later.
- The add-on remembers your presets between NVDA sessions.
- If you change your mind while saving, renaming, or deleting, you can cancel the dialog before confirming.

## Accessibility

Voice Switcher is intended for blind and low-vision NVDA users. The dialog is keyboard-friendly and works with standard NVDA navigation and speech feedback.

## Compatibility

This add-on is intended for NVDA `2024.1` and later.
